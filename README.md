# JapanGo - 일본어 단어 학습 앱

React + TypeScript + Vite로 개발된 일본어 단어 학습 애플리케이션입니다.

## 🚀 실행 방법

1. **의존성 설치**
   ```bash
   npm install
   ```

2. **개발 서버 실행**
   ```bash
   npm run dev
   ```

3. **브라우저에서 확인**
   - 개발 서버가 실행되면 브라우저에서 `http://localhost:5173` 주소로 접속하세요.

## 📁 프로젝트 구조

```
japango-app/
├── public/
│   └── vocabulary/         # 주차별 단어 데이터 (JSON 파일)
│       ├── week1.json
│       ├── week2.json
│       └── ...
├── src/
│   ├── App.tsx            # 메인 React 컴포넌트
│   ├── main.tsx           # React 진입점
│   ├── index.css          # 글로벌 스타일
│   └── App.css            # 컴포넌트 스타일
└── index.html             # HTML 템플릿
```

## 🎯 주요 기능

- **📚 주차별 학습**: 1-6주차 단어 체계적 학습
- **🃏 카드 모드**: 단어 카드 뒤집기, 힌트, 자동재생
- **📝 테스트 모드**: 다양한 문제 유형 (한자→뜻, 뜻→한자, 읽기→뜻)
- **🔊 음성 재생**: 일본어 발음 듣기
- **❌ 틀린 문제 복습**: 오답 노트 기능
- **📊 학습 통계**: 진행률 및 성취도 추적
- **🌙 다크 모드**: 라이트/다크 테마 전환

## 🔧 문제 해결

만약 화면에 아무것도 표시되지 않는다면:

1. **브라우저 개발자 도구 확인** (F12)
   - Console 탭에서 오류 메시지 확인
   - Network 탭에서 파일 로딩 상태 확인

2. **포트 확인**
   - 다른 애플리케이션이 5173 포트를 사용 중일 수 있습니다.
   - Vite가 자동으로 다른 포트를 할당할 경우 터미널에서 확인

3. **캐시 정리**
   ```bash
   # 개발 서버 중지 후
   npm run build
   npm run preview
   ```
